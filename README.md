Navaja is a multi-platform app for booking barber appointments.

## Features

- User Authentication
- User Profile Management
- Service Booking

## Core Technologies

- Angular
- Expo / React Native
- tRPC
- Firebase

## Libraries

- Redux
- Luxon
- RxJS
- Zod
- Google APIs
- Tailwind

## Monorepo Management

- NPM Workspaces
- Turborepo
